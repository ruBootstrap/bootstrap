Аккордеон

Создавайте вертикально сворачивающиеся аккордеоны в сочетании с нашим JavaScript плагином Collapse.

Как это устроено

Аккордеон использует внутри collapse, чтобы сделать его сворачиваемым. Для рендеринга раскрытого аккордеона добавьте класс .open в .accordion.

Эффект анимации этого компонента зависит от медиазапроса prefers-reduced-motion. См. раздел с уменьшенным движением в нашей документации по специальным возможностям.
