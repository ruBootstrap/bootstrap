Баги браузеров

В настоящее время Bootstrap обходит множество нерешённых багов в основных браузерах, чтобы обеспечить наилучший кроссбраузерный опыт. Некоторые баги, как перечисленные ниже, мы не можем решить самостоятельно.

Мы публично публикуем баги браузеров, которые влияют на нас, чтобы ускорить их исправление. Для информации о совместимости Bootstrap с браузерами смотрите документацию по совместимости.

Смотрите также:

  • Chromium issue 536263: [meta] Проблемы, влияющие на Bootstrap
  • Mozilla bug 1230801: Исправить проблемы, влияющие на Bootstrap
  • WebKit bug 159753: [meta] Проблемы, влияющие на Bootstrap
  • Обходные решения багов браузеров в jQuery
