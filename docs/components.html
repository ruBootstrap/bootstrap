Выпадающие меню

Пример

Выпадающее, контекстное меню для отображения списков ссылок. Сделано интерактивным с помощью плагина JavaScript для выпадающих меню.

  • Действие
  • Другое действие
  • Что-то еще здесь
  • Разделенная ссылка
<ul class="dropdown-menu" role="menu" aria-labelledby="dropdownMenu">
  <li><a tabindex="-1" href="#">Действие</a></li>
  <li><a tabindex="-1" href="#">Другое действие</a></li>
  <li><a tabindex="-1" href="#">Что-то еще здесь</a></li>
  <li class="divider"></li>
  <li><a tabindex="-1" href="#">Разделенная ссылка</a></li>
</ul>

Разметка

Рассмотрим только выпадающее меню, вот необходимая HTML. Вам нужно обернуть триггер и выпадающее меню в .dropdown, или другой элемент, который объявляет position: relative;. Затем просто создайте меню.

<div class="dropdown">
  <!-- Link or button to toggle dropdown -->
  <ul class="dropdown-menu" role="menu" aria-labelledby="dLabel">
    <li><a tabindex="-1" href="#">Действие</a></li>
    <li><a tabindex="-1" href="#">Другое действие</a></li>
    <li><a tabindex="-1" href="#">Что-то еще здесь</a></li>
    <li class="divider"></li>
    <li><a tabindex="-1" href="#">Разделенная ссылка</a></li>
  </ul>
</div>

Опции

Выравнивание меню вправо и добавление дополнительных уровней выпадающих меню.

Выравнивание меню

Добавьте .pull-right к .dropdown-menu, чтобы выровнять выпадающее меню вправо.

<ul class="dropdown-menu pull-right" role="menu" aria-labelledby="dLabel">
  ...
</ul>

Отключенные опции меню

Добавьте .disabled к <li> в выпадающем меню, чтобы отключить ссылку.

  • Обычная ссылка
  • Отключенная ссылка
  • Другая ссылка
<ul class="dropdown-menu" role="menu" aria-labelledby="dropdownMenu">
  <li><a tabindex="-1" href="#">Обычная ссылка</a></li>
  <li class="disabled"><a tabindex="-1" href="#">Отключенная ссылка</a></li>
  <li><a tabindex="-1" href="#">Другая ссылка</a></li>
</ul>

Подменю на выпадающих меню

Добавьте дополнительный уровень выпадающих меню, появляющийся при наведении, как у OS X, с несколькими дополнительными изменениями в разметке. Добавьте .dropdown-submenu к любому li в существующем выпадающем меню для автоматического стиля.

Группы кнопок

Примеры

Две основные опции, а также две более специфические вариации.

Одна группа кнопок

Оберните серию кнопок с .btn в .btn-group.

<div class="btn-group">
  <button class="btn">Left</button>
  <button class="btn">Middle</button>
  <button class="btn">Right</button>
</div>

Несколько групп кнопок

Объедините наборы <div class="btn-group"> в <div class="btn-toolbar"> для более сложных компонентов.

<div class="btn-toolbar">
  <div class="btn-group">
    ...
  </div>
</div>

Вертикальные группы кнопок

Сделайте набор кнопок появляться вертикально, а не горизонтально.

<div class="btn-group btn-group-vertical">
  ...
</div>

Варианты с чекбоксами и радиокнопками

Группы кнопок также могут функционировать как радиокнопки, где может быть активна только одна кнопка, или как чекбоксы, где может быть активна любая кнопка. См. документацию по JavaScript для этого.

Навигация: вкладки, табы и списки

Легкие дефолты Тот же синтаксис, разные классы

Все компоненты навигации здесь—вкладки, табы и списки—делят один базовый синтаксис и стили через .nav класс.

Базовые вкладки

Возьмите обычный <ul> ссылок и добавьте .nav-tabs:

  • Home
  • Profile
  • Messages
<ul class="nav nav-tabs">
  <li class="active">
    <a href="#">Home</a>
  </li>
  <li><a href="#">...</a></li>
  <li><a href="#">...</a></li>
</ul>

Базовые табы

Возьмите тот же HTML, но используйте .nav-pills вместо:

  • Home
  • Profile
  • Messages
<ul class="nav nav-pills">
  <li class="active">
    <a href="#">Home</a>
  </li>
  <li><a href="#">...</a></li>
  <li><a href="#">...</a></li>
</ul>

Состояние отключенных элементов

Для любого компонента навигации (вкладки, табы или список), добавьте .disabled для серых ссылок и отсутствия эффектов при наведении. Ссылки останутся кликабельными, однако, если вы не удалите href атрибут. В качестве альтернативы вы можете реализовать пользовательское JavaScript для предотвращения этих кликов.

  • Clickable link
  • Clickable link
  • Disabled link
<ul class="nav nav-pills">
  ...
  <li class="disabled"><a href="#">Home</a></li>
  ...
</ul>

Выравнивание компонентов

Чтобы выровнять ссылки навигации, используйте .pull-left или .pull-right классы утилит. Оба класса добавят CSS float в указанном направлении.

Настраиваемые

Поскольку вкладки и табы по умолчанию горизонтальны, просто добавьте второй класс, .nav-stacked, чтобы они выглядели вертикально.

  • Messages

  ...
</ul>

Настроенные табы

  • Home

Таббируемая навигация

Придайте жизнь вашим вкладкам с простым плагином для переключения между контентом через вкладки. Bootstrap интегрирует таббируемые вкладки в четыре стиля: сверху (по умолчанию), справа, снизу и слева.

Пример таббируемой навигации

Чтобы сделать вкладки таббируемыми, создайте .tab-pane с уникальным ID для каждой вкладки и заключите их в .tab-content.

  • Section 1
  • Section 2
  • Section 3

I'm in Section 1.

Howdy, I'm in Section 2.

What up girl, this is Section 3.

<div class="tabbable"> <!-- Only required for left/right tabs -->
  <ul class="nav nav-tabs">
    <li class="active"><a href="#tab1" data-toggle="tab">Section 1</a></li>
    <li><a href="#tab2" data-toggle="tab">Section 2</a></li>
  </ul>
  <div class="tab-content">
    <div class="tab-pane active" id="tab1">
      <p>I'm in Section 1.</p>
    </div>
    <div class="tab-pane" id="tab2">
      <p>Howdy, I'm in Section 2.</p>
    </div>
  </div>
</div>

Вкладки с затуханием

Чтобы сделать вкладки затухающими, добавьте .fade к каждому .tab-pane.

Требуется плагин jQuery

Все таббируемые вкладки питаются нашим легким плагином jQuery. Подробнее о том, как придать жизнь таббируемым вкладкам, можно прочитать на странице документации по JavaScript.

Навигационный бар

Базовый навигационный бар

Чтобы начать, навигационные панели статичны (не фиксированы в верхней части) и поддерживают название проекта и основную навигацию. Поместите один в любом месте внутри .container, который устанавливает ширину вашего сайта и контента.

Title
  • Home
  • Link
  • Link
<div class="navbar">
  <div class="navbar-inner">
    <a class="brand" href="#">Title</a>
    <ul class="nav">
      <li class="active"><a href="#">Home</a></li>
      <li><a href="#">Link</a></li>
      <li><a href="#">Link</a></li>
    </ul>
  </div>
</div>

Компоненты навигационного бара

Бренд

Простая ссылка, чтобы показать название вашего проекта или имени только требуется тег <a>.

Title 
<a class="brand" href="#">Project name</a>

Навигационные элементы просты в добавлении через неупорядоченные списки.

  <li><a href="#">Link</a></li>
</ul>

Вы можете легко добавить разделители между вашими навигационными ссылками, добавив пустой элемент списка и простой класс. Просто добавьте это между ссылками:

Формы

Чтобы правильно стилизовать и расположить форму в навигационном баре, добавьте соответствующие классы, как показано ниже. Для обычной формы включите .navbar-form и либо .pull-left, либо .pull-right, чтобы правильно выровнять ее.

<form class="navbar-form pull-left">
  <input type="text" class="span2">
  <button type="submit" class="btn">Submit</button>
</form>

Форма поиска

Для более настроенной формы поиска добавьте .navbar-search к form и .search-query к input для специальных стилей в навигационном баре.

<form class="navbar-search pull-left">
  <input type="text" class="search-query" placeholder="Search">
</form>

Выравнивание компонентов

Выровняйте навигационные ссылки, форму поиска или текст, используя .pull-left или .pull-right классы утилит. Оба класса добавят CSS float в указанном направлении.

Зачем использовать миниатюры

Миниатюры (ранее .media-grid до версии 1.4) отлично подходят для сеток фотографий или видео, результатов поиска изображений, розничных товаров, портфолио и многого другого. Они могут быть ссылками или статическим контентом.

Простая, гибкая разметка

Разметка миниатюр проста—ul с любым количеством li элементов достаточно. Также она очень гибкая, позволяя вам добавить любой тип контента с немного большей разметкой для обертки вашего контента.

Использование классов сетки

Наконец, компонент миниатюр использует существующие классы системы сетки—например, .span2 или .span3—для контроля размеров миниатюр.

Разметка

Как уже упоминалось ранее, необходимая разметка для миниатюр проста и прямая. Вот пример базовой настройки для ссылок на изображениях:

<ul class="thumbnails">
  <li class="span4">
    <a href="#" class="thumbnail">
      <img data-src="holder.js/300x200" alt="">
    </a>
  </li>
  ...
</ul>

Для пользовательского HTML-контента в миниатюрах разметка изменяется незначительно. Чтобы позволить блочному контенту в любом месте, мы меняем <a> на <div> вот так:

<ul class="thumbnails">
  <li class="span4">
    <div class="thumbnail">
      <img data-src="holder.js/300x200" alt="">
      <h3>Надпись миниатюры</h3>
      <p>Подпись миниатюры...</p>
    </div>
  </li>
  ...
</ul>

<button type="button" class="close" data-dismiss="alert">&times;</button>

Примеры и разметка

Базовые

Полоски

Добавьте .active к .progress-striped, чтобы анимировать полоски слева направо. Не поддерживается во всех версиях IE.

<div class="progress progress-striped active">
