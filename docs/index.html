Создан для всех и везде.

Bootstrap делает фронтенд-разработку быстрее и проще. Он подходит для людей любого уровня, устройств любых форм и проектов любого масштаба.

Препроцессоры

Bootstrap поставляется с обычным CSS, но исходный код использует два самых популярных CSS-препроцессора: Less и Sass. Быстро начните с готового CSS или работайте с исходниками.

Один фреймворк — любое устройство.

Bootstrap легко и эффективно масштабирует ваши сайты и приложения с единой кодовой базой — от телефонов до планшетов и десктопов с помощью CSS-медиа-запросов.

Много возможностей

С Bootstrap вы получаете подробную и красивую документацию по основным HTML-элементам, десятки пользовательских HTML и CSS-компонентов и отличные jQuery-плагины.

Bootstrap — это open source. Он размещён, развивается и поддерживается на GitHub.
