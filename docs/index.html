Знакомьтесь, Bootstrap.

Хотите узнать, почему стоит выбрать Bootstrap? Вот несколько причин.

От разработчиков — для разработчиков.

Создан в Twitter @mdo и @fat. Bootstrap использует LESS CSS, компилируется с помощью Node и управляется через GitHub, чтобы помочь разработчикам делать крутые вещи в вебе.

Для всех и каждого.

Bootstrap отлично выглядит и работает не только в современных десктопных браузерах (а также в IE7!), но и на планшетах и смартфонах благодаря адаптивному CSS.

Много возможностей.

12-колоночная адаптивная сетка, десятки компонентов, JavaScript-плагины, типографика, элементы форм и даже онлайн-кастомизация, чтобы сделать Bootstrap по-настоящему вашим.

Сайты на Bootstrap.

Для ещё большего количества сайтов на Bootstrap посетите неофициальный Tumblr или посмотрите примеры.
