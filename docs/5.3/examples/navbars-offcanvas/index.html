Панель навигации offcanvas с примерами

В этом примере показано, как адаптивные меню вне холста работают на панели навигации. Для размещения панелей навигации ознакомьтесь с примерами top и fixed top.

Сверху вниз вы увидите темную панель навигации, светлую панель навигации и адаптивную панель навигации, каждая из которых имеет встроенные элементы offcanvas. Измените размер окна браузера до большой контрольной точки, чтобы увидеть переключатель для offcanvas.
