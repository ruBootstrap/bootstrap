Установка через менеджер пакетов

Установите исходные файлы Sass и JavaScript Bootstrap через npm, RubyGems, Composer или Meteor. Установки через менеджер пакетов не включают документацию или наши полные скрипты сборки. Вы также можете использовать наш npm шаблон репозитория для быстрого создания проекта Bootstrap через npm.

{{ highlight (printf ("npm install bootstrap@%s") .Site.Params.current_version) "sh" "" }} {{ highlight (printf ("gem install bootstrap -v %s") .Site.Params.current_ruby_version) "sh" "" }}

Читайте нашу документацию по установке для получения дополнительной информации и других менеджеров пакетов.
