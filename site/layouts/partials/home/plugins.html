Мощные JavaScript плагины без jQuery

Добавляйте переключаемые скрытые элементы, модальные окна и offcanvas меню, всплывающие подсказки и многое другое — все без jQuery. JavaScript Bootstrap ориентирован на HTML, что означает, что большинство плагинов добавляются с помощью атрибутов data в вашем HTML. Нужно больше контроля? Подключайте отдельные плагины программно.
