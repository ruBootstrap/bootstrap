Настройте все с помощью Sass

Bootstrap использует Sass для модульной и настраиваемой архитектуры. Импортируйте только те компоненты, которые вам нужны, включите глобальные опции, такие как градиенты и тени, и напишите свой собственный CSS с нашими переменными, картами, функциями и миксинами.

Включите весь Sass Bootstrap

Импортируйте один таблицу стилей и вы готовы к работе со всеми функциями нашего CSS.

{{ highlight (printf `// Variable overrides first $primary: #900; $enable-shadows: true; $prefix: "mo-"; // Then import Bootstrap @import "../node_modules/bootstrap/scss/bootstrap"; `) "scss" "" }}

Включите то, что вам нужно
